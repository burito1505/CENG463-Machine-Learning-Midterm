# CENG 463 - Machine Learning Midterm Project

## Description
This repository contains the solutions for the CENG 463 Machine Learning Midterm Project. The project is divided into 5 independent tasks, each focusing on a core machine learning paradigm:
1. **Regression:** Regularisation, Feature Engineering, and Model Selection.
2. **Classification:** Handling Imbalanced Data, Ensemble Methods, and Cost-Sensitive Learning.
3. **Dimensionality Reduction:** PCA, t-SNE, and Manifold Learning transformations.
4. **Clustering:** Gaussian Mixture Models, Cluster Ensembles, and Stability Analysis.
5. **Neural Networks:** Deep Learning architectures (MLP, Custom CNN), Transfer Learning (VGG-16), Interpretability (LIME), and Adversarial Robustness (FGSM).

## Installation
To ensure full reproducibility, a `requirements.txt` file is provided. It is recommended to create a virtual environment first, and then install the necessary dependencies by running:

```bash
pip install -r requirements.txt
```

## How to Run
Each question's solution is contained within its own Jupyter Notebook (`.ipynb`) file. To evaluate the code:
1. Ensure your environment has the dependencies installed.
2. Open the respective notebook (e.g., `question5_neural_networks.ipynb`, `california_housing_regression.ipynb`).
3. Execute the cells sequentially from top to bottom. 

*Note: Some notebooks (especially Question 5) perform model training. If you lack a GPU, training might take a few minutes.*
